# 📜 罗德岛协议助手

**一个专为《明日方舟》卫戍协议活动打造的 AstrBot 辅助插件**

## ✨ 功能特性

- 📚 **盟约图鉴** – 查看指定盟约下的所有干员及其等阶，支持自动发送本地攻略图片
- 👤 **干员详情** – 查询干员的特质、等阶、所属盟约
- 🎵 **金曲推荐** – 随机推送 B 站金曲视频链接
- 🎲 **卫一把** – 随机推荐 2~3 个盟约，帮你决定下一把玩法

---

## 📦 安装

### 方式一：通过 AstrBot 插件市场安装（推荐）

在 AstrBot 管理面板中搜索 `罗德岛协议助手` 或 `rhodes_island_guard` 并安装。

### 方式二：通过 URL 安装

1. 复制插件仓库地址：`https://github.com/kkky-yyyk/astrbot_plugin_rhodes_island_guard`
2. 在 AstrBot WebUI 插件页面，点击右下角 `+` → **通过 URL 安装**，粘贴地址并确定。

## 📝 使用方法

### 基础命令

所有命令均以 / 开头（可在 WebUI 中自定义命令前缀），支持私聊和群聊。

### 命令表

|  命令    |     说明     |
|:--------:|:--------------|
| `/盟约 <盟约名>` | 查看该盟约下的干员列表，若盟约为核心盟约会额外发送本地攻略图片 |
| `/干员 <干员名>` | 查询干员的特质、等级、所属盟约 |
| `/金曲` | 随机推送一首已保存的 B 站金曲 |
| `/卫一把` | 随机推荐 2~3 个盟约 |


## 📁 自定义攻略图片

当用户查询 核心盟约 时，插件会自动发送一张本地图片。你可以通过以下方式替换或添加图片：

1. 在插件目录下创建 images 文件夹（若没有）
2. 将图片放入该文件夹，命名为 <核心盟约>.png（也支持 .jpg、.jpeg、.gif）(如“叙拉古.png”)
3. 重载插件后生效
如果你想为其他盟约也添加图片，可以修改 covenant_query 方法中的发送逻辑，使其尝试发送任意盟约名对应的图片。
